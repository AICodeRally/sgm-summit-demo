# Dispute

## Purpose
Central hub for dispute intake, case management, and SLA-driven resolution.

## When to use
Use this page when a dispute or exception needs review, escalation, or SLA monitoring.

## How to use
1. Go to Cases Management to review open cases and priorities.
2. Use Case SLA & Load to monitor SLA risk and workload.
3. Open Case Analytics for trends and bottlenecks.
4. Use Quick Actions to submit cases or jump to escalations.

## Key data
Active case counts, SLA thresholds, priority levels, and resolution timelines.
