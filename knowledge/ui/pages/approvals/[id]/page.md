# Approvals / Detail

## Purpose
Provide the full approval packet so approvers can make an auditable decision.

## When to use
Use this page when you need to approve, reject, or request changes for a specific item.

## How to use
1. Review the request summary, linked document/plan, and policy citations.
2. Check evidence attachments and SLA timing.
3. Record a decision and route to the next approver if required.

## Key data
Approval status, SLA deadline, approver chain, evidence links, and audit note.
