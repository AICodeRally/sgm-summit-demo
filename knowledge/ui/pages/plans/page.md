# Plans

## Purpose
Provide lifecycle management for compensation plans, including drafts, approvals, and publishing.

## When to use
Use this page to create a new plan, edit an existing plan, or track plan status.

## How to use
1. Create a plan from scratch or from a template.
2. Add sections, calculations, and policy-aligned clauses.
3. Submit for approval and monitor status through publication.

## Key data
Plan status, version history, approval state, and associated documents.
