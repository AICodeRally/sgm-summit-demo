# Reports

## Purpose
Create standard governance reports for audits, committees, and operational reviews.

## When to use
Use this page to export coverage summaries, approval logs, or compliance snapshots.

## How to use
1. Choose the report type (coverage, approvals, compliance).
2. Set filters for date range, tenant, or committee.
3. Export or share the generated report.

## Key data
Report type, time range, approval status, and compliance metrics.
