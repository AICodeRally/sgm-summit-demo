# Operate

## Purpose
Run daily governance workflows, including document lifecycle, plan changes, approvals, and operational reporting.

## When to use
Use this page to publish or review documents, manage plans, clear approvals, or track operational alerts.

## How to use
1. Open Document Library to review, version, or publish governance documents.
2. Use Plans Management for plan creation and updates.
3. Work the Approvals Queue to keep SGCC/CRB decisions on time.
4. Check Calendar, Reports, Search & Compare, and Notifications for daily visibility.

## Key data
Document inventory, approval counts, active plan totals, and notifications sourced from operational APIs.
