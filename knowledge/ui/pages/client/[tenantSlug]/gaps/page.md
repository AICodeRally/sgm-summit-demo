# Client / Gaps

## Purpose
Surface gaps that need remediation to meet governance standards.

## When to use
Use this page to prioritize remediation work for a client.

## How to use
1. Review gaps sorted by severity and policy.
2. Open a gap to see recommended remediation.
3. Assign owners and target dates for closure.

## Key data
Gap severity, policy references, remediation status, and owners.
