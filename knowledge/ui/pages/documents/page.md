# Documents

## Purpose
Central library for frameworks, policies, procedures, templates, and guides with status and version tracking.

## When to use
Use this page to locate a document, check its status, compare versions, or validate approvals.

## How to use
1. Filter by document type or status on the left panel.
2. Search by code, title, or category.
3. Select a document to view details, versions, and actions.
4. Use the demo filter to focus on demo or real documents.

## Key data
Document code, type, status, version, owner, and last-updated metadata.
