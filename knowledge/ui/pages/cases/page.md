# Cases

## Purpose
Manage disputes and exceptions across the organization.

## When to use
Use this page to review, prioritize, and update cases.

## How to use
1. Filter cases by status or priority.
2. Open a case to review evidence and history.
3. Update status or route for approval.

## Key data
Case status, priority, SLA, and owner.
